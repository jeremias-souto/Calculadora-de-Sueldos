# Calculadora de Sueldos

## Descripción
**Calculadora de Sueldos** es una aplicación web sencilla e intuitiva diseñada para facilitar el cálculo del salario total de un trabajador. Permite ingresar múltiples variables que componen el sueldo de un empleado, incluyendo:
- Precio y cantidad de horas trabajadas.
- Precio y cantidad de horas extras.
- Extras o bonos por presencialidad.

Además, cuenta con una funcionalidad práctica que desglosa el sueldo total calculado en porcentajes de "sueldo en blanco" (formal/registrado) y "sueldo en negro" (informal/no registrado), basándose en el porcentaje ingresado por el usuario.

## Tecnologías Usadas
Este proyecto está construido íntegramente con tecnologías web estándar, sin dependencias externas:
- **HTML5**: Estructura de la aplicación.
- **CSS3**: Estilos visuales.
- **JavaScript (Vanilla)**: Lógica de cálculo numérico e interacción del Document Object Model (DOM).

## Cómo ejecutarlo
Dado que es una aplicación del lado del cliente totalmente estática, no necesitas instalar NodeJS, bases de datos ni ningún servidor web complejo. 

Sigue estos sencillos pasos:
1. Descarga o clona en tu computadora los archivos de este proyecto.
2. Ve a la carpeta raíz del proyecto (`Calculadora de Sueldos`).
3. Haz doble clic en el archivo `index.html`. 
4. El programa se abrirá automáticamente en tu navegador web predeterminado (por ejemplo: Chrome, Firefox, Edge).
5. Ingresa los datos solicitados en cada campo y presiona el botón **"Calcular"** para ver los resultados en la parte inferior de la pantalla.
